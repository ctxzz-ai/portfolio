About

WebGLアーティスト / フロントエンドエンジニア

ブラウザで実現できる表現の限界を追求し、インタラクティブなプロジェクトを多数制作。 Three.jsをはじめとしたWebGLフレームワークを得意とし、演出・UI・UXまでトータルで設計します。

モーションデザイン、シェーダープログラミング、パフォーマンスチューニングを組み合わせ、ビジネス課題を解決する魅力的な体験を提供します。

  • Three.js / WebGL 2.0
  • GLSL Shader Programming
  • Procedural Animation
  • Creative Coding / Generative Art
  • React / TypeScript / Web Components

Works

WebGL & Three.js Works

iframeによる埋め込み、またはモーダルで拡大表示できるインタラクティブな作品を紹介します。 各プロジェクトはダミーデータで構成されており、簡単に差し替え可能です。

Nebula Flow

パーティクルで描く銀河空間。マウス操作に反応して軌跡が変化する没入感の高いデモ。

Tidal Prism

フラクタルノイズで生成した海原。マテリアルを調整し、幻想的な光の屈折を表現。

Synth City

低ポリゴンのビル群をリズミカルにライトアップ。音楽連動演出のプロトタイプ。
